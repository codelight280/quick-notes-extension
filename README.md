# Quick Note Browser Extension

A simple, elegant browser extension for taking quick notes anywhere.

![Extension Screenshot](icons/icon128.png)

## Features ‚ú®
- **Simple Input Box**: Clean, focused writing space
- **Auto-Save**: Automatically saves as you type
- **Manual Save**: Click save button or use Ctrl+S
- **Clear Option**: Remove all text with confirmation
- **Keyboard Shortcuts**: 
  - `Ctrl+S` - Save note
  - `Ctrl+K` - Clear note
  - `Esc` - Close extension
- **Persistent Storage**: Notes are saved between sessions

## Installation üöÄ

### Method 1: Load Unpacked (Development)
1. Download or clone this repository
2. Open Chrome/Edge/Brave browser
3. Go to `chrome://extensions/`
4. Enable **Developer mode** (toggle in top-right)
5. Click **"Load unpacked"**
6. Select the `quick-note-extension` folder
7. Pin the extension to your toolbar

## How to Use üìù
1. Click the extension icon in your toolbar
2. Type your note in the text area
3. Your note auto-saves as you type
4. Use buttons or keyboard shortcuts to manage your note
5. Close the popup - your note will be there when you return
